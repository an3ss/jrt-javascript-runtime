Class: PromptBox

jrt.PromptBox()

Implements a prompt dialog box with a message, an optional title, an input field, and two buttons labelled "Ok" and "Cancel". If the user accepts, the return value is the input element's value. Otherwise it's null.

This class must be instanced with the jrt.SystemDialogBox.prompt method.
